# Software-Development-Training

软件开发实训--期刊管理系统

## 选题

期刊管理系统：

1. 数据分析（如统计读者信息、期刊借阅情况、期刊出入库情况等）
2. 用户管理（三类用户：系统管理员、期刊管理员、级别不同的读者，系统用户各种权限管理，如读者用户的申请、注册、读者权限，查找）
3. 读者管理（包括读者信息的增删改查）
4. 期刊管理（包括期刊的基本信息的增、查，如刊号、年、期、数量、相应的文章）
5. 借阅管理（查询、借阅、归还、预约）
